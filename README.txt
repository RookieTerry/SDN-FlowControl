源文件功能:
- entity/Bucket.java 组表项中的Bucket
- entity/FlowEntry.java 定义了流表项中必要的字段，以及设置、获取字段的方法
- entity/GroupEntry.java 定义了组表项中必要的字段，以及设置、获取字段的方法
- sdn/Add.java 创建流表项，并添加流表到交换机
- sdn/Delete.java 删除流表项
- sdn/GroupAdd.java 添加组表项，并添加组表到交换机
- sdn/List.java 显示流表项
- tools/Constant.java 定义了实验中用到的常量
- tools/FlowTableTool.java 定义了向指定交换机添加流表项、清空流表项的操作
- tools/HttpRequest.java 定义了发送HTTP请求到控制器的指定URL用到的GET、POST方法
